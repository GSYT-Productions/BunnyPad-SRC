This is for the various official icon packs that can be used in BunnyPad. If you want to make your own iconpack, please do the following:
Name the icon images as follows:
  "selectall.png"
  "share.png"
  "support.png"
  "team.png"
  "undo.png"
  "wordwrap.png"
  "charmap.png"
  "copy.png"
  "cut.png"
  "delete.png"
  "document.png"
  "exit.png"
  "find.png"
  "font.png"
  "info.png"
  "new.png"
  "open.png"
  "paste.png"
  "pdf.png"
  "printer.png"
  "redo.png"
  "replace.png"
  "save.png"
  "saveas.png"
  "cake.png"
  "nocake.png"
put those in a "images/" directory, compress the directory INTO A ZIP FILE FORMAT, give the zip file a name that describes the iconpack, create a fork and submit a pull request.
if you use icons you did not make, please provide the source/artist in a "about.txt" within the icon pack.
